About Rust

Rust, the programming language, is an exciting new language that aims to be safe, concurrent, and practical.

It combines the familiar C style syntax that uses curly braces with the features you'd expect in a modern language.

Rust hopes to "preserve large-system integrity, availability and concurrency" by the separation it creates between its abstractions and implementations.

Design Patterns

Design patterns have made a resurgence in recent years with the publication of the 1994 seminal work called Design Patterns: Elements of Reusable Object-Oriented Software (also called the Gang of Four book). It provided a catalog of design patterns that could be used across software systems.

This website takes all the design patterns in the Gang of Four book and turns them into working examples using the language Rust.

Sources

As was mentioned, Design Patterns: Elements of Reusable Object-Oriented Software is the main source of information provided on this website.

Yet the Gang of Four book isn't the most approachable for newcomers. A more gentle introduction is the Head First Design Patterns book. It provides very memorable examples with easier to understand explanations.

Contributing

Rust Design Patterns is completely available as open source. It is hosted in GitHub and can be forked by anyone.

If you have suggestions on how to make these design patterns better, fork it and make a pull request with your updates. If the updates are awesome (which they probably will be) then the pull requests will be merged into the main project.
